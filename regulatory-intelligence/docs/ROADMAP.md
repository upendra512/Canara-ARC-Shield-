# Development Roadmap

## Phase 1: Learn LangGraph Basics

- Keep node functions simple.
- Print state after each node.
- Compile and invoke the intake graph.
- Write one test per node contract.

## Phase 2: Build Layer 0

- Validate local document paths.
- Extract text from text PDFs.
- Add OCR for scanned PDFs.
- Extract basic metadata.
- Store documents locally.

## Phase 3: Build Local Retrieval

- Create chunk schema.
- Add local embedding model.
- Build FAISS index.
- Retrieve related documents with metadata filters.

## Phase 4: Build Layer 1 Intelligence

- Add classification rules.
- Add clause extraction.
- Add obligation extraction.
- Add transparent risk categorization.

## Phase 5: Build Layer 2 Compliance Intelligence

- Add controls and policies.
- Map obligations to controls.
- Identify gaps.
- Generate recommendations for human review.

## Phase 6: Build Human Review

- Add review queue.
- Add approval workflow.
- Add final repository.
- Add audit trail.

## Phase 7: Harden Offline Deployment

- Add Docker.
- Add database migrations.
- Add backup and restore.
- Add role-based access.
- Add evaluation reports.
